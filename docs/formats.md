# Supported Formats

RedactB4X converts uploaded documents to Markdown before running detection. This lets it handle a wide range of formats through one processing pipeline.

| Format | Extension |
|--------|-----------|
| PDF | `.pdf` |
| Word Document | `.docx` |
| Plain Text | `.txt` |
| CSV | `.csv` |
| JSON | `.json` |
| HTML | `.html` |
| Markdown | `.md` |
| EPUB | `.epub` |
| Excel | `.xlsx` |
| PowerPoint | `.pptx` |
